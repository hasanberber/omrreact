# OMR Form Uygulaması

Bu uygulama, optik işaret tanıma (OMR) formları oluşturmanıza ve okumanıza olanak tanır.

## Özellikler

- ✅ OMR formları oluşturma
- ✅ Formları özelleştirme (soru sayısı, seçenek sayısı, vb.)
- ✅ Cevap anahtarı belirleme
- ✅ Form okuma (görsel yükleme veya kamera)
- ✅ Otomatik puanlama
- ✅ Tesseract.js ile OCR desteği

## Kurulum

```bash
# Bağımlılıkları yükleyin
npm install

# Uygulamayı başlatın
npm start
```

Uygulama [http://localhost:3000](http://localhost:3000) adresinde açılacaktır.

## Üretim İçin Build

```bash
npm run build
```

## Kullanım

1. **Form Oluşturucu**: Özel OMR formları tasarlayın
2. **Form Okuyucu**: Doldurulmuş formları okuyun ve puanlayın
3. **Kamera Desteği**: Mobil cihazlarda kamera ile form tarama

## Teknolojiler

- React 18
- Tesseract.js (OCR)
- Lucide React (İkonlar)
- Tailwind CSS
